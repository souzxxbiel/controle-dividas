# Site para Anotar Devedores Automático Vou criar um site completo em um único arquivo HTML para gerenciar devedores de forma simples e automática. ```html Devedores Manager - Controle de Dívidas

Devedores Manager

Controle de dívidas de forma simples e automática. Basta digitar o nome e o valor!

Adicionar Novo Devedor

Lista de Devedores

Total Devido: R$ 0.00

Nenhum devedor cadastrado

Adicione seu primeiro devedor usando o formulário acima

``` ## Recursos e Funcionalidades do Site 1. **Interface moderna e responsiva** - Design limpo com gradientes, cards e elementos visuais agradáveis 2. **Adição rápida de devedores** - Basta digitar nome e valor e clicar em "Salvar Devedor" 3. **Lista de devedores completa** - Exibe nome, valor devido e data/hora do registro 4. **Cálculo automático do total** - Mostra o valor total devido por todos os devedores 5. **Pesquisa em tempo real** - Filtra a lista de devedores conforme você digita 6. **Edição de registros** - Clique no ícone de edição para carregar um devedor no formulário 7. **Exclusão de registros** - Remova devedores com confirmação 8. **Armazenamento local** - Os dados são salvos no navegador (localStorage) 9. **Notificações** - Feedback visual para ações realizadas 10. **Totalmente responsivo** - Funciona em dispositivos móveis e desktop ## Como Usar 1. Preencha o nome do devedor e o valor devido 2. Clique em "Salvar Devedor" para adicionar à lista 3. Use a barra de pesquisa para encontrar devedores específicos 4. Clique no ícone de lápis para editar um devedor existente 5. Clique no ícone de lixeira para remover um devedor Os dados são automaticamente salvos no seu navegador, então você pode fechar e reabrir a página que as informações permanecerão salvas.
